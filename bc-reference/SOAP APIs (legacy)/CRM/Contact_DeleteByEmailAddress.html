Contact_DeleteByEmailAddress

Deletes a Contact record based on customer's email address.

Warning: This API endpoint will eventually be deprecated (existing implementations will still continue to work). We strongly advise to use the new CRM v3 REST API endpoints. Here is an example on how to delete customer record using the new endpoints.

Request

  • Method: SOAP
  • Server: https://[app key here]-[site_ID here]-apps.worldsecuresystems.com. Take a look at the Authorize your API calls document for more info on how this URL is formed.
  • Path: /catalystwebservice/catalystcrmwebservice.asmx
  • The username and password fields:You can make SOAP API calls can be authorized either by using the actual username or password of an Admin username or, if you are making the calls from an app leave the username field empty and use the authorization token as the password.

Parameters

  • {% raw %}siteId{% endraw %} - ID of the site (integer)
  • {% raw %}username{% endraw %} - email address of user account, leave empty if using site token (string)
  • {% raw %}password{% endraw %} - password of user account, or site authentication token for specified site (string)
  • {% raw %}email{% endraw %} - (string)

To use a site token instead of username/password, send an empty {% raw %}username{% endraw %} field and the site token as the {% raw %}password{% endraw %}. See example below.

Response

A Contact_DeleteByEmailAddressResponse object with the following properties:

  • {% raw %}Contact_DeleteByEmailAddressResult{% endraw %} - (boolean)

Examples

Accepts and returns XML as Content-Type.
