Registration (buy) layout

Is used by these modules

  • This layout is used for the Check Out form. Users get to this form as the final step of the shopping experience after clicking the checkout button outputted by the {tag_buybutton} tag found on the shopping cart layout.

Accessible from

  • Admin Console:Â Site Manager > Module Templates > Online Shop Layouts
  • SFTP & Develop Mode: /Layouts/OnlineShop/order_registration-XX.html

Replace "XX" with the country code you have added in the Site Settings > Site Domains panel (see the "Country" column).

If you have multiple domains with different "Country" values you will find multiple order-registration-XX files, one for each country. This way, with one layout for every country defined you can vary the contents of the layout to accomodate a multilingual website.
