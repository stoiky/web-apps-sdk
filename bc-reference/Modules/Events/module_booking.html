{module_booking}

Description

Displays Booking items.

This module also supports Liquid, take a look at this article to learn how to access its Liquid output tags and render it using your own custom logic.

  • New named parameters documentation
  • Legacy module documentation

Syntax

{% raw %}{module_booking render="item|collection" filter="item" itemid="neventId or classificationId" rowCount="10" collection="my_custom_collection_name" template="/folder/template.tpl"}{% endraw %}

Note: When using named parameters always wrap the individual parameters in double quotes. Also, do note that modules using named parameters do not work for campaigns just yet.

Parameters

  • render - possible values are item or collection (the default is item). The render parameter indicates whether the template content is used to render a collection or to render a single item. Take a look at the render parameter's article for more details on how to use it
  • filter - filtering criteria for display and can be one of the following:
    • item - individual item
    • all - all items
    • latest - latest items
    • random - random item
    • randomClassified - displays a random item in a particular category
    • latestClassified - latest items in a category
    • classified - all the items in a category
  • itemid - depending on the filter chosen this can be either one of (see the Examples section for details):
    • the ID of the booking module
    • the ID of the category
    • the number of booking items to be displayed
  • rowCount - number of items per row
  • collection - use this if you need to access the module's Liquid output tags
  • template - optional, for this module this can either represent the path of the custom template you want to use, or use a blank value (template="") if you need to suppress the original output and use Liquid output tags instead

This module can be rendered using:

  • the list layout - on this layout you can use either the legacy tags or Liquid output tags. Here is a list of all the tags you can use the Event layout reference page
  • your own custom layout - take a look at the custom templates article for more details
  • Liquid collections - here are the Liquid output tags you can use:

Examples

  • {% raw %}{module_booking filter="all"}{% endraw %} - displays all the bookings
  • {% raw %}{module_booking filter="item" itemid="24407"}{% endraw %} - displays an individual event item with the ID 24407
  • {% raw %}{module_booking filter="l" itemid="5"}{% endraw %} - displays 5 latest events
  • {% raw %}{module_booking filter="random"}{% endraw %} - displays a single random item
  • {% raw %}{module_booking filter="classified" itemid="14075"}{% endraw %} - displays all the items classified in the category with the ID 14075
  • {% raw %}{module_booking filter="randomClassified" itemid="14075"}{% endraw %} - displays a random item classified in the category with the ID 14075
  • {% raw %}{module_booking filter="item" itemid="63522" template="/layouts/custom/bookings.tpl"}{% endraw %} - displays an individual item using a custom template
  • {% raw %}{module_booking filter="all" collection="collection1" template=""}{% endraw %} - in this case nothing will be displayed, however the module's Liquid tags will become available in the collection1 collection.

Syntax

{% raw %}{module_booking,filter,ID or numberOfBookings}{% endraw %}

Parameters

  • filter - filtering criteria for display and can be one of the following:
    • i - individual item
    • a - all items
    • l - latest items
    • r - random item
    • cr - displays a random item in a particular category
    • cl - latest items in a particular category
  • ID or numberOfBookings
    • id - the ID of the booking module. This is system generated and does not need to be changed
    • numberOfBookings - can be used in conjunction with l parameter to limit the number of bookings displayed

Layouts

  • Event Layouts > List Layout
  • This module also supports custom templates

Examples

  • {% raw %}{module_booking,a}{% endraw %} - displays all the bookings
  • {% raw %}{module_booking,i,24407}{% endraw %} - displays an individual event item with the ID 24407
  • {% raw %}{module_booking,l,5}{% endraw %} - displays 5 latest events
  • {% raw %}{module_booking,r}{% endraw %} - displays a single random item
  • {% raw %}{module_booking,c,14075}{% endraw %} - displays all the items classified in the category with the ID 14075
  • {% raw %}{module_booking,cr,14075}{% endraw %} - displays a random item classified in the category with the ID 14075
  • {% raw %}{module_booking,i,63522 template="/layouts/custom/bookings.tpl"}{% endraw %} - displays an individual item using a custom template
