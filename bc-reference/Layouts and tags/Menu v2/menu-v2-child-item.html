Module menu V2 - children.html

Is used by these modules

  • {module_menu}

Accessible from

Unlike other layouts the module_menu v2 layouts can only be accessed in the "Develop" section of the site or if you connect via sFTP at this path - /ModuleTemplates/Menu/MENU_NAME/children.html

This is the Module Template used to customize the individual menu items.


Renders these tags

{% raw %}{tag_menuchilditem}{% endraw %}This will render a list of all menu items with the same parent
{% raw %}{tag_menuitemdescription}{% endraw %}Item Description
{% raw %}{tag_menuitemtargetframe}{% endraw %}Target Frame e.g. _blank
{% raw %}{tag_menuitemimageurl}{% endraw %}Image URL
{% raw %}{tag_menuitemimagerolloverurl}{% endraw %}Image URL for the "rollover" state
{% raw %}{tag_menuitemimageselectedurl}{% endraw %}Image URL for the "selected" state
{% raw %}{tag_menuitemcssclass}{% endraw %}CSS class name
{% raw %}{tag_menuitemcssrolloverclass}{% endraw %}CSS class name for the "rollover" state
{% raw %}{tag_menuitemcssselectedclass}{% endraw %}CSS class name for the "selected" state
{% raw %}{tag_menuitemidname}{% endraw %} Menu item ID
{% raw %}{tag_menuitemlabel}{% endraw %} Menu item label
{% raw %}{tag_menuitemurl}{% endraw %} Menu item URL e.g. /contact.html
{% raw %}{tag_menuitemcssclass_withclass}{% endraw %}CSS class name, with  class attribute e.g. class="itemClass"
{% raw %}{tag_menuitemcssrolloverclass_withclass}{% endraw %}CSS class name  for the "rollover" state, with class attribute e.g.  class="itemClassRollover"
{% raw %}{tag_menuitemcssselectedclass_withclass}{% endraw %}CSS class name  for the"selected" state, with class attribute e.g.  class="itemClassSelected"
{% raw %}{tag_menuitemidname_withid}{% endraw %}Item ID, with id attribute  e.g. id="myId"
